# Doxygen Awesome (vendored)

Assets from **[doxygen-awesome-css](https://github.com/jothepro/doxygen-awesome-css)** **v2.4.2** (MIT).

| File | Role |
|------|------|
| `doxygen-awesome.css` | Base theme |
| `doxygen-awesome-sidebar-only.css` | Sidebar layout |
| `doxygen-awesome-sidebar-only-darkmode-toggle.css` | Toggle placement with sidebar |
| `doxygen-awesome-darkmode-toggle.js` | Light / dark switch |
| `doxygen-awesome-fragment-copy-button.js` | Copy on code hover |
| `doxygen-awesome-paragraph-link.js` | Permalink on headings |
| `doxygen-awesome-interactive-toc.js` | TOC highlight + progress (needs `PAGE_OUTLINE_PANEL = NO`) |

Scripts are loaded from `docs/doxygen/header.html`. Branding: `../custom.css`, `../logo.svg` (`PROJECT_LOGO` in `docs/Doxyfile.in`).
